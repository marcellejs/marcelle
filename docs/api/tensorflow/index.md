# @marcellejs/tensorflow

This package integrates [Tensorflow.js](https://www.tensorflow.org/js) with Marcelle, making it possible to run inference or train TFJSmodels in Marcelle client-side apps.

## Installation

To install the package:

::: code-group

```bash [npm]
npm install @marcellejs/tensorflow
```

```bash [yarn]
npm add @marcellejs/tensorflow
```

```bash [pnpm]
pnpm add @marcellejs/tensorflow
```

:::
